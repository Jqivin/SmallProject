# PictrueView
**1**、有菜单、工具条，支持快捷键；

**2**、通过文件对话框打开需要访问的图片*(持单选和多选功能)

**3**、打开的图片可以在该软件中央显示

**4**、含有对图像进行灰度化((R+G+B)/3)、二值化(阈值滑条控件可调)、3×3均值滤波、伽马变换(**针对彩图**)、**1**种边缘检测等功能

**5**、软件支持中文、英文两种语言，用户可自由切换，设计并美化用户界面

**6**、可执行文件需要有自己定义的名称、自己设计的图标

**7、需利用消息提示对话框，并含有“关于本软件对话框(标注自己姓名学号)”，消息对话框图标要求自己定制**
